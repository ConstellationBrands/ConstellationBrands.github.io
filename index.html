Welcome to Constellation Brands Product Locator Public API!
 

The goal of this API is to provide developers access to our product location information in an open and easy-to-use manner.

The API provides the following information about our products:

  • Brand Name
  • Sub-Brand Name
  • Varietal (Flavor)
  • Appellation/Region
  • Bottle/Packaging Size
  • In-Store Locations
  • On-Premise/Restaurant Locations 

The Product Locator API provides a common interface allowing other applications to access Product Locator related data. The framework is based on open-standard REST based web service technologies which allow for maximum flexibility and compatibility with other systems. The API functionality can only be used by authorized customers who have the valid API Key. Below are the list of functions that are exposed through the API.



API Calls  Description
Brands Retrieves the list of all active Brand Names along with their Brand Codes.
Brand Details Retrieves specific information about an individal brand like Sub Brand, Varietal, Size, etc.
Item Details Retrieves specific information about an individual item like Appellation, Description, EAN, GTIN, etc.
State Restrictions Retrieves all the State Codes along with their restrictions.
Product Locator Search Retrieves the search results (Store locations) for the given query parameters which includes proximity search as well.
